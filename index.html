Oddit

Oddit is a Ruby on Rails restaurant and bar rating app where users can post and comment about their favorite local spots. With profile creating functionality a user can create a profile, post a venue, rate it, and comment on other posts as well. I've programmed these posts to only be editable by the poster, and allow other users to post photos of the venue as well.

Unslctd Advce

Unsolicited Advice is a Ruby on Rails random quote generating application that randomly displays user-submitted quotes upon refreshing the page. Users can create an account, post a new quote, and refresh to see new pieces of unsolicited advice.

Dive

Dive is a Ruby on Rails user-submitted online learning platform where users can both teach and learn video classes. With credit card processing abilities, users can purchase classes and enroll, which unlocks all lessons and videos.

Cultivate

Cultivate is a Ruby on Rails single-page application, where users can sign up and create a list of all of the goals they want to accomplish to improve their life. Cultivate allows the user to submit a goal, display it on the page, and the user can cross off the goal as it is completed.

Favor Booth

For the E-Commerce brand Favor, this is a tradeshow booth that I designed from the ground up for the Architectural Digest Tradeshow in New York. The display is a series of wood panels with laser cut holes to replicate a halftone, which has shelving and pegs for product. The pink is pink felt, creating a variied texture throughout the booth.
